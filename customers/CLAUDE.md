# Customers Context

## Overview
Each customer has their own subdirectory with a `CLAUDE.md` covering account context, active work, and history.

## Customer Directory Structure
Each customer folder should contain:
- `CLAUDE.md` — account context, contacts, active work
- `audits/` — audit results and findings (optional)
- `deliverables/` — anything produced for the customer (optional)

## Adding a New Customer
1. Copy `_template.md` as `[customer-name]/CLAUDE.md`
2. Fill in account context
3. Add to the table above
